2D Procedural Generation

A small prototype for generating terrain on a 2D tilemap and optimising it for single thread peformance.

Project Overview

This project is a technical demo developed to gain a deeper understanding of optimising large algorithms and creating complex game systems. The system focuses on generating fully procedural 2D tile worlds using Perlin noise, biomes and chunk based rendering and loading. The objective was to create a scalable and modular architecture capable of generating entire maps while maintaining smooth peformance on the main thread.

The tile generation uses layerd Perlin noise and fall off maps to produce near natural elevation and temperature variations. To visually see the distinction between elevation and temperature the values are then processed through a biome lookup system, which asigns different biomes depending on the elevation and temperature values of the tile. To improve peformance the values are generated before tile placement. The tile placement function runs in a co-routine to avoid freezing the game on the main thread. Chunks are used to allow for the option to use Occlusion Culling to avoid rendering all tiles for the player during runtime.

A key focus was achieving readable and maintainable code, allowing for further biomes and systems to be integrated in the future. The demo can be easily implemented into a future game a tweaked in the Unity Inspector for various needs. The project also explores island generation usiing fall off curves to genrate circular landmasses.

Features & Techniques

  • Biome Mapping System — a flexible lookup table defines biomes types based on elevationa dn temperature thresholds, allowing for near seamless biome transitions.
  • Optimisation — world generation is handled through co-routines and chunks, distributing the generation over multiple frames to maintain smooth peformance without multithreading.
  • Scalable — by focusing on modular structure and being to able to change the generation behaiour in the inspector has created a project that has excellent foundations for future features to build upon.
